# خطة بناء منصة إدارة التعلم (LMS) - مركز الخليج للوقاية من الأمراض ومكافحتها (GCDC)

> مبنية على تحليل العرض الفني المقدم من شركة نسيج لنظام D2L Brightspace
> تاريخ الإعداد: 2026-03-31

---

## 1. ملخص تنفيذي

بناء منصة LMS مخصصة بالكامل تحقق نفس الوظائف المطروحة في عرض D2L Brightspace مع ميزات إضافية تشمل:
- دعم **1000 متعلم نشط** (قابل للتوسع)
- دعم كامل **للغة العربية** (RTL)
- استضافة سحابية (SaaS)
- تكامل مع أنظمة السجلات (SOR)
- هوية بصرية مطابقة لمعايير GCDC

## 3. البنية التقنية (Technology Stack)

### Frontend (واجهة المستخدم)
| التقنية | السبب |
|---------|-------|
| **Next.js 14+ (App Router)** | SSR/SSG، أداء عالي، SEO |

| **Tailwind CSS** | تصميم سريع ومتجاوب مع دعم RTL |

| **React Query (TanStack)** | إدارة حالة البيانات من الخادم |
| **Zustand** | إدارة الحالة المحلية |
| **next-intl** | دعم تعدد اللغات (عربي/إنجليزي) |

### Backend (الخادم)
| التقنية | السبب |
|---------|-------|
| **Node.js + NestJS** | بنية قوية وقابلة للتوسع |
| **TypeScript** | توحيد اللغة مع الواجهة |
| **PostgreSQL** | قاعدة بيانات علائقية موثوقة |
| **Prisma ORM** | إدارة قاعدة البيانات بسهولة |

### البنية التحتية
| التقنية | السبب |
|---------|-------|
| **AWS (أو Azure)** | استضافة سحابية موثوقة (مراكز بيانات في الخليج) |

| **AWS S3** | تخزين الملفات والمحتوى |
| **CloudFront CDN** | توزيع المحتوى بسرعة |
| **GitHub Actions** | CI/CD أتمتة النشر |

### الأمان
| التقنية | السبب |
|---------|-------|

## 4. الوحدات الوظيفية (Modules)

### 4.1 إدارة المستخدمين والصلاحيات
- تسجيل المستخدمين (يدوي + رفع جماعي CSV)
- أدوار متعددة: **مدير النظام، مدير التدريب، المدرب، المتعلم**
- ربط المستخدمين بالأقسام والإدارات (الهيكل التنظيمي)
- SSO مع أنظمة المؤسسة
- إدارة أولياء الأمور/المشرفين (Guardian relationships)
- تسجيل وإلغاء تسجيل تلقائي

### 4.2 إدارة الدورات والمحتوى
- إنشاء وإدارة الدورات التدريبية
- دعم أنماط التعلم المتعددة:
  - **تعلم ذاتي** (Self-paced)
  - **تعلم متزامن** (Live/Synchronous)
  - **تعلم مختلط** (Blended)
- محرر محتوى غني (Rich Text Editor)
- رفع ومعاينة: فيديو، PDF، صور، عروض تقديمية
- تنظيم المحتوى في وحدات ودروس (Modules & Lessons)
- قوالب دورات جاهزة
- نسخ واستيراد/تصدير الدورات

### 4.3 التقييم والاختبارات
- إنشاء اختبارات متنوعة (اختيار متعدد، صح/خطأ، مقالي، مطابقة)
- بنك أسئلة مركزي
- تصحيح تلقائي ويدوي
- تحديد مواعيد ومدة الاختبارات
- محاولات متعددة وإعدادات مرنة
- تقارير نتائج تفصيلية

### 4.4 التقارير والتحليلات
- لوحة معلومات رئيسية (Dashboard)
- تقارير تقدم المتعلمين
- تقارير إكمال الدورات
- تحليلات النشاط والمشاركة
- تقارير قابلة للتصدير (Excel, PDF)
- مؤشرات أداء رئيسية (KPIs)

### 4.5 الشهادات والإنجازات
- إصدار شهادات إتمام تلقائية
- تصميم قوالب شهادات مخصصة
- شارات رقمية (Digital Badges)
- سجل إنجازات المتعلم

### 4.6 الإشعارات والتواصل
- إشعارات فورية (Push Notifications)
- بريد إلكتروني تلقائي
- رسائل داخلية بين المستخدمين
- إعلانات على مستوى الدورة أو النظام
- تقويم الأحداث والمواعيد

### 4.7 التكاملات (Integrations)
- **LTI 1.3** لتكامل أدوات خارجية
- **SCORM 1.2 / 2004** لاستيراد محتوى تعليمي
- **xAPI (Tin Can)** لتتبع التعلم المتقدم
- تكامل مع أنظمة السجلات (SOR) عبر:
  - CSV/SFTP (رفع يدوي أو تلقائي)
  - REST API
- تكامل مع أنظمة HR
- تكامل مع Microsoft Teams / Zoom للفصول الافتراضية

### 4.8 الهوية البصرية والتخصيص
- تطبيق ألوان GCDC بالكامل
- شعار وهوية المركز
- تخصيص متعدد المستويات (مستوى النظام، القسم، الدورة)
- صفحة تسجيل دخول مخصصة
- دعم كامل لـ RTL

## 10. مقارنة: البناء الذاتي vs شراء D2L Brightspace

| المعيار | بناء ذاتي | D2L Brightspace (نسيج) |
|---------|----------|----------------------|
| **التحكم الكامل** | ✅ تحكم 100% | ❌ محدود بإمكانيات المنصة |
| **التخصيص** | ✅ بلا حدود | ⚠️ محدود |
| **التكلفة الأولية** | ⬆️ أعلى (تطوير) | ⬇️ أقل (اشتراك) |
| **التكلفة المستمرة** | ⬇️ أقل على المدى الطويل | ⬆️ اشتراك سنوي مستمر |
| **وقت الإطلاق** | ⬆️ 6 أشهر | ⬇️ 2-3 أشهر |
| **الملكية الفكرية** | ✅ ملك للمؤسسة | ❌ ملك لـ D2L |
| **الصيانة** | ⚠️ فريق داخلي مطلوب | ✅ يتولاها المزود |
| **التكاملات** | ✅ مرنة بالكامل | ⚠️ محدودة بما يوفره D2L |
| **الاستضافة** | ✅ تحكم في موقع البيانات | ⚠️ AWS (حسب D2L) |
| **الدعم الفني** | ⚠️ فريق داخلي | ✅ دعم من نسيج و D2L |

## 12. الخلاصة

بناء منصة LMS مخصصة يوفر **تحكماً كاملاً** و**ملكية فكرية** و**مرونة غير محدودة** في التخصيص والتطوير المستقبلي. مع خطة تنفيذ مدتها **24 أسبوعاً (6 أشهر)** وفريق من **9 أفراد**، يمكن بناء منصة تضاهي D2L Brightspace مع مميزات إضافية تلبي احتياجات GCDC بدقة.

النظام المقترح يدعم:
- ✅ 1000+ متعلم نشط
- ✅ دعم كامل للعربية (RTL)
- ✅ تكامل SOR (CSV + API)
- ✅ LTI 1.3 + SCORM
- ✅ هوية GCDC البصرية
- ✅ تدريب متعدد الأنماط
- ✅ تقارير وتحليلات متقدمة
- ✅ شهادات وإنجازات رقمية
